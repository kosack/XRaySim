XRaySim - simulation of 3DXDM experiment.  See
documentation in xraysim\Documentation\xraysim.ps


USES:
	libsvl         (vector library, included in source)
	openGL         (should be installed on system already)
	Fltk	       (graphical widget library for interface, included)



To back up the source code, go to the command prompt,
switch to the xraysim directory and type:

	archive <date>

ex: archive 4-10-99  -
(will produce xraysim-arch-4-10-99.zip)
